jquery-grade 评分插件

该插件可任意设置最大等级，默认最大等级是5 。 支持设置指定默认等级

兼容IE8+,Foxfire,Chrome 等主流浏览器

参数

id //唯一标识

title //评分标题

num //指定等级

maxnum //最大等级

complete //回调函数 fun(num){}

使用方式

引入 jquery.js

引入 cygrade.min.css

引入 cygrade.min.js 

$(function(){
	$("#cygrade").cygrade({
		title:'酒店星级->',
		complete:function(t){
			alert("选中的星级:"+t);
		}
	});
});
